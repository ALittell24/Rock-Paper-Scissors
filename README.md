# Rock Paper Scissors

## Overview

A Python implementation of the classic Rock Paper Scissors game where the user plays against a computer-generated opponent.

## Features

- User versus computer gameplay
- Randomized computer choices
- Win, lose, and tie conditions
- Command-line interface

## Technologies Used

- Python

## Concepts Practiced

- Conditional statements
- Random number generation
- User input handling
- Program logic

## What I Learned

This project helped me develop experience working with randomization, decision logic, and game outcome evaluation. It reinforced my understanding of conditional statements and user interaction within a Python application.
